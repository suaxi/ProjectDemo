#   02 项目完成情况
---
---

## 工作完成情况

**第一天：** 用户：登录，注册

**第二天：** 用户：查看商品，按不同方式搜索商品（模糊搜索）

**第三天：** 用户：购买商品

**第四天：** 管理员：登录，查看管理员端的商品信息

**第五天：** 管理员：添加商品，修改商品信息

**第六、七天：** Web前端开发
